# SmartBudget üöÄ

**SmartBudget** is a modern, high-performance personal finance application designed to help users track expenses, manage budgets, and visualize financial habits with clarity and precision.

SmartBudget combines powerful analytics with a premium, fluid user interface to make financial management effortless and engaging.

## ‚ú® Key Features

### üìä Advanced Analytics
- **Interactive Dashboards**: Powered by `recharts`, giving you deep insights into spending patterns.
- **PowerBI-Style Visuals**: Custom-built chart containers with focus modes, data export, and drill-down capabilities.
- **Real-Time Updates**: Instant reflection of new expenses and budget adjustments.

### üí∞ Comprehensive Budgeting
- **Smart Budget Planning**: Create monthly budgets for specific categories (Rent, Food, Entertainment).
- **Progress Tracking**: Visual progress bars with status indicators (On Track, Over Budget) and color-coded alerts.
- **Monthly Breakdown**: Detailed "Deep Dive" view to analyze spending month-over-month.

### üîê Secure & Seamless
- **Authentication**: Secure login/signup flow powered by **Firebase Auth**.
- **Cloud Sync**: All data is securely stored and synced in real-time using **Firebase Firestore**.
- **Responsive Design**: Fully responsive UI that works across desktop, tablet, and mobile devices.

## üöÄ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v16.0.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HeyBoY-ops/Capstone2.git
   cd Capstone2
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root directory and add your Firebase config keys:
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173` to view the application.
